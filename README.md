Installation
------
- Clone the repository:
`git clone https://github.com/sunnyg1210/adminbsbmaterial-ANGULAR`

- Install npm packages:
`npm install`

- Run Angular dev server:
`ng serve`

- Build Production code:
`ng build --prod`
